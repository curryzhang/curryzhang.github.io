# SignalR介绍

Asp.net SignalR是微软为实现实时通信的一个类库。一般情况下，任何一个技术的出现都是为了解决特定的问题。SignalR也不例外。

我们知道网站开发中最常见到的一个功能就是在线聊天室。 现在的网上的大部分在线聊天室都是基于2种技术：

  1. AJAX
  2. Flash

聊天室要解决最大的问题就是 消息的推送。当N个在线用户 同时加入一个聊天室时，1个用户发送消息，服务端就要把这个消息转发给特定的人。

之前的技术都是通过Javascript来不停地发送请求来轮训 服务端的新的消息。这种定期发送Ajax请求给服务器的方式，在用户很大的情况下给服务器带来很大的压力。

WebSockets这个技术的出现，很好地解决了这个问题，恰恰支持可以主动推送消息，SignalR 支持WebSockets。

另外SignalR开发的程序实现跨平台部署在Linux环境下。

# 使用场景

SignalR既然是为实时而生的，这样就决定了其使用场所。具体适用情景有如下几点：

  • 聊天室，如在线客服系统，IM系统等
  • 股票价格实时更新
  • 消息的推送服务
  • 游戏中人物位置的实时推送
