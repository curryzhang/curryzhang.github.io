    # RabbitMQ学习1--安装

    首先Rabbit MQ 是建立在强大的Erlang OTP平台上，因此安装Rabbit MQ的前提是安装Erlang

    1. 下载并安装 Erlang OTP For Windows (vR16B03)
    2. 运行安装 Rabbit MQ Server Windows Installer (v3.2.3)
    3. 使用Rabbit MQ 管理插件，可以更好的可视化方式查看Rabbit MQ 服务器实例的状态，打开cmd，切换到安装目录（C:\Program Files (x86)\RabbitMQ Server\rabbitmq_server-3.2.3\sbin），输入命令激活：
    rabbitmq-plugins enable rabbitmq_management

    接着我们需要重启服务

    net stop RabbitMQ && net start RabbitMQ

    这时候可能会发生错误：发生系统错误 5。拒绝访问。

    解决方法：以管理员身份运行cmd

    输入网址，打开监控页面： http://localhost:15672 (默认账号和密码：guest 和guest)

    # 配置RabbitMQ用户权限

    配置新用户和权限，继续打开cmd命令，cd到安装目录sbin下

    # 查询服务状态

    rabbitmqctl status

    运行rabbitmqctl status报错： 网上给的解决有很多，主要是三点：

    1. 重新安装服务
    • 执行rabbitmq-service remove
    • 执行rabbitmq-service install
    1. 更正Erlang的cookie

    检查C:\Windows.erlang.cookie 及 C:\用户\用户名.erlang.cookie这两个文件是否一致，如果不一致用其中一个替换另一个即可 3. 修改RabbitMQ服务的登录身份 将RabbitMQ服务的登录账户换成Administrator,然后重启服务。

    # 列举虚拟主机列表

    rabbitmqctl list_vhosts

    # 列举用户列表

    rabbitmqctl list_users

    # 添加用户

    rabbitmqctl  add_user  test 123asd

    # 设施权限

    >rabbitmqctl  set_permissions  test  ".*"  ".*"  ".*"

    # 分配用户组

    rabbitmqctl  set_user_tags test administrator

    # 删除用户

    rabbitmqctl delete_user test

    # 修改用户名密码

    rabbitmqctl change_password test 123123
