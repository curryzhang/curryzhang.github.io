# 让你的网站支持MarkDown

使用方法

  1. 后端Markdig
  2. 使用tui-editor，参考vue-element-admin
  3. 使用showdown，可以使用js或者yarn add showdown

具体使用：

<textarea v-model="content"></textarea>
<div v-html="html"/>

import showdown from 'showdown'
export default {
    data() {
        return {
            content:"",
            html:'',
        }
    },
    components: {
        showdown
    },
    methods:{
        converntHtml(){
            var converter =newshowdown.Converter();
            this.html=converter.makeHtml(this.content);  
        }
    }
}
</script>

但是这边不支持table，解决方案安装showdown-table
