# 8、Redis的常用命令

连接到服务器

redis-cli -h host -p port -a password

测试服务器是否连接成功

ping

设置密码

config set requirepass ****

验证密码

auth ***

获取密码

config get requirepass

查看所有的key

keys *

根据条件查询key

keys pattern

pattern表示查询条件，可以实现模糊查询

  • *通配任意多个字符，keys * 表示返回当前redis数据库中所有的key。
  • ? 通配任意一个字符，
  • [] 通配[]内的某一个字符

示例

set name1 hello
set name2 curry
set name3 blues

keys na* //返回name1,name2,name3
keys name?//返回name1,name2,name3
keys name[2345]//返回name2,name3

查看key的类型

type key

删除key

del key1 key2 ... keyn

判断key是否存在

exists key

随机获取一个key

randomkey

查询key的生命周期

ttl key

不存在key返回-2，永久存在返回-1，否则返回key的有效期（单位：秒）

设置key的有效期

expire key 秒数

设置key永久有效

persist key

给key重命名

rename key newkey

如果newkey已经存在，那么newkey会覆盖掉

示例：

set name1 hello
set name2 curry

rename name1 name2

get name1//返回空
get name2//返回hello

给key重命名

renamenx key newkey

当newkey不存在的时候，则改名，返回1，否则不改名，返回0。

关闭连接

quit

清空缓存

flushall
