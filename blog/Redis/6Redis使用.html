# 6、C#使用Redis

Redis官网提供了很多开源的C#客户端，如下图： 这边我们采用ServiceStack.Redis来举例说明

安装nuget包ServiceStack.Redis

static RedisClient redisClient = new RedisClient("127.0.0.1", 6379);//redis服务IP和端口
//如果设置了密码
//static RedisClient redisClient = new RedisClient("127.0.0.1", 6379, "***");

static void Main(string[] args)
{
    Console.WriteLine(redisClient.Get<string>("name"));
    Console.ReadKey();
}
